This is the main repository for the second version of OayasTech's CBT software (CBT-V2), you can find the developnment (staging) instance of this web app [here](https://nextjs.org), the project is a NextJS web application bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Contributing to the Project

If you are contributing to this software, after cloning this repository run the devlopment server using any of the following commands:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
